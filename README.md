
# WemersonDev -> Ticket App

A React Native App (Only configured to IOS) to control Ticket flow.

Project created during a course at RocketSeat.

Stack used to create the app :

React Native
- Expo - https://expo.dev

- Firebase - https://firebase.google.com 
    (firestore and auth)
